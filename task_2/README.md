
## Проверка знаний

**Написать программу для рисования на экране при помощи цветного шара:**
- Захватить цветной шар
- На экране с демонстрацией текущего положения шара или в отдельном окне рисовать линию по положению центра шара
- Предусмотреть кнопку для стирания
- Чем ближе шар тем толще рисуемая линия
