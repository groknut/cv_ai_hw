
# CVML

## Распознавание текста при помощи k-Nearest Neighbour

Используя KNN перевести изображения содержащие строки в текст. Учесть порядок символов, пробелы и символы, который состоят из двух элементов. Программа должна вывести номер изображения и распознанный текст.

Во вложенном архиве, в корне находятся изображения для распознавания, в папке train изображения отдельных символов для извлечения признаков.
